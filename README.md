# CS546A Fianl Project

A Hospital Managment System which provides patient and doctor a convenient way to access to hospital services

## Features
#### For Patients:
* User Registration/Login
* Doctor appointment booking
* Prescription checkout
#### For Doctors:
* User Registration/Login
* Prescription system
* Medicine purchasing

## Project Structure
* Front-end
* `view`
* Server
`routes`
* Database
* `data`
